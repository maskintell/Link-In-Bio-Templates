# Profile Image Instructions

1. Place your professional photo here and name it `avatar.jpg`
2. For best results, use a square image (1:1 aspect ratio)
3. Recommended size: 400x400 pixels (will be displayed at 90x90)
4. Optional: Create a high-resolution version `avatar@2x.jpg` for retina displays

If you don't have an image, you can:
- Use a placeholder service: https://ui-avatars.com/api/?name=Your+Name
- Use an Unsplash URL in the HTML instead
